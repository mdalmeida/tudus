# Tudús

App de productividad personal — mix entre TickTick, Notion y Google Keep.

## Stack
- React + Tailwind
- Backend: Notion / Google Sheets (intercambiable)
- Auth: Google / Meta

## Estructura
```
tudus/
├── src/
│   └── App.tsx          # Prototipo React completo
├── docs/
│   ├── PRD.md           # Definición de producto
│   └── ANALYSIS.md      # Análisis funcional y técnico
└── README.md
```
